# 04. 2점 레벨링 캡처 기반 축 정렬 자동 회전 보정 (바탕 문서)

> 고지: 본 문서는 기술/전략 참고자료이며 법률 자문이 아닙니다.  
> 기준 문서: [PATENT_PORTFOLIO_ANALYSIS_KR_PCT.md](../PATENT_PORTFOLIO_ANALYSIS_KR_PCT.md)

## 문서 목적
2점 입력으로 회전 보정을 자동 수행하는 레벨링 기능을 실무 설명용으로 정리한다.  
핵심은 "사용자가 어떤 순서로 입력하면 어떤 보정이 반영되는가"를 명확히 만드는 것이다.

## 기능 정의
이 기능은 레벨링 캡처 상태에서 두 점을 선택하면, 두 점의 기울기 각도를 계산해 수평 또는 수직 기준축에 맞도록 회전값을 자동 조정한다.  
Pinpoint(개별/전역), Compare, Analysis 모드에 맞춰 대상 회전 상태를 갱신한다.

핵심 포인트:
1. 레벨링 상태머신(active/mode/target/points/axis)
2. `atan2` 기반 각도 계산과 axis별 delta 계산
3. 모드별 회전 상태에 자동 반영 후 캡처 종료

## 사용방법
대표 시나리오:
1. 레벨링 기능을 켜고 축을 선택한다(수평 또는 수직).
2. 첫 번째 점을 선택해 기준점을 고정한다.
3. 두 번째 점을 선택해 기준선을 지정한다.
4. 자동 보정된 회전값이 적용되었는지 확인한다.
5. 필요 시 다시 두 점을 찍어 미세 조정한다.

운영 팁:
1. 수직 보정은 중심선이 아닌 실제 기준 에지(기둥/패턴선)에서 점을 찍는 편이 정확하다.
2. 회전값 표시 UI를 함께 확인하며 반복 보정하면 검증이 쉽다.

## 결과
정성 효과:
1. 수작업 회전 슬라이더 조정 대비 보정 속도가 빨라진다.
2. 작업자 간 보정 편차가 줄어든다.

관찰 포인트(측정 후보):
1. 목표 정렬 도달까지 클릭 횟수
2. 보정 후 잔여 각도 오차(도 단위)
3. 동일 샘플 반복 측정 시 표준편차

## 강점
1. 계산 절차가 단순하고 직관적이라 교육 비용이 낮다.
2. 축(수평/수직) 선택으로 다양한 도메인 장면에 대응 가능하다.
3. 모드별 회전 상태를 공통 로직으로 처리해 유지보수가 쉽다.
4. 실시간 오버레이로 사용자 피드백이 명확하다.

## 특허 문서 전환을 위한 작성 포인트
권리화 핵심 구성요소:
1. 2점 입력 기반 각도 산출 단계
2. 선택 축 기준 오차(delta) 계산 단계
3. 모드별 대상 회전 반영 단계

차별 주장 포인트:
1. 단순 각도 측정이 아니라 자동 보정까지 포함한 상호작용 루프
2. 공통 UI 동작을 모드별 회전 상태로 매핑하는 구조

실험/증빙으로 보강할 요소:
1. 수동 회전 대비 시간 단축률
2. 반복 보정 시 오차 안정성 데이터

## 작성 시 주의사항/한계
1. 기준점 입력 품질에 결과가 영향을 받으므로 사용자 입력 오차를 전제로 설명해야 한다.
2. 자동 보정이 항상 최종 완료를 의미하지는 않으며 2차 미세조정이 필요할 수 있다.
3. 특허 작성 시 "2점 입력"과 "축 기반 delta 반영"의 결합을 빠뜨리면 범위가 약해진다.
