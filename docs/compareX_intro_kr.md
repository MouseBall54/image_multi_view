# compareX 프로그램 소개

## 간략 개요
- **compareX**는 다중 이미지를 동시에 비교·정렬·분석·후처리할 수 있는 Electron 기반 데스크톱 앱이자 웹 데모입니다.
- Compare / Pinpoint / Analysis 세 가지 전문 모드를 통해 워크플로우에 맞는 작업 환경을 즉시 전환할 수 있습니다.
- 모든 연산이 로컬에서 수행되어 이미지가 외부 서버로 전송되지 않으며, 오프라인에서도 동일한 기능을 제공합니다.
- React + TypeScript + OpenCV.js 조합으로 고성능 UI와 필터 체인을 지원해, 대용량 이미지도 실시간으로 다룰 수 있습니다.

## 핵심 강점
1. **완전한 프라이버시**  
   - 100% 클라이언트 사이드 처리, 네트워크 전송 없음, 민감 데이터도 안심하고 사용할 수 있습니다.
2. **전문가용 Viewing Mode**  
   - Compare: 최대 6×4 레이아웃과 지능형 파일 매칭으로 대량 비교를 자동화합니다.  
   - Pinpoint: 참조점, 개별 스케일·회전, 정밀 레벨링으로 정렬 작업을 정교하게 다룹니다.  
   - Analysis: 60+ 필터 체인을 구성하고 실시간 파라미터 조정으로 분석 효율을 높입니다.
3. **고급 필터 시스템**  
   - OpenCV 가속 필터 19종(이상)과 커스텀 필터 체인, 필터 라벨링/프리셋, 다중 미리보기 등 전문 기능을 포함합니다.
4. **유연한 UI·단축키**  
   - 파워유저를 위한 20+ 키보드 단축키, 미니맵/그리드/레이아웃 토글, 고해상도 스크린샷 도구 등 생산성을 극대화합니다.
5. **TIFF 및 대용량 이미지 지원**  
   - UTIF 기반 TIFF 디코딩과 효율적 메모리 캐싱으로 일반 뷰어에서 다루기 어려운 파일도 안정적으로 처리합니다.

## 차별화 포인트
- **통합 워크스테이션**: 뷰어·정렬·분석·필터링을 하나의 UI에서 해결해 툴 전환 비용이 없습니다.
- **지능형 파일 매칭**: 다중 폴더를 읽고 이름/확장자 규칙 기반으로 자동 정렬해 반복 작업을 크게 줄입니다.
- **정교한 정렬 도구**: 핀포인트 모드의 레벨링, 참조점, Shift/Swap 드래그 방식 등은 사진 감리·품질검사에 특화된 기능입니다.
- **실시간 필터 체인 미리보기**: 필터 순서를 직관적으로 표시하고, 전/후 비교를 병렬 배치해 실험 시간을 단축합니다.
- **오프라인 우선 설계**: Electron 패키지와 자동 업데이트, 로컬 데이터 보존을 기본값으로 제공해 민감 환경에 적합합니다.

## 사용법 요약
1. **설치/실행**
   - 데스크톱 앱: 최신 패키지를 설치하거나 `npm install && npm run dev`로 로컬 빌드합니다.
   - 웹 데모: 브라우저에서 [https://mouseball54.github.io/image_multi_view/](https://mouseball54.github.io/image_multi_view/) 접속.
2. **모드 선택**
   - 상단 모드 스위처에서 Compare / Pinpoint / Analysis 중 목적에 맞는 모드를 선택합니다.
3. **이미지 불러오기**
   - 폴더 선택, 드래그 앤 드롭, 개별 뷰어 지정 등 다양한 입력 방식을 지원합니다.
4. **작업 진행**
   - Compare: 자동 매칭 결과를 기반으로 동기화된 줌/팬, 레이아웃 전환, 이미지 토글을 활용합니다.
   - Pinpoint: 참조점 배치, 개별 스케일·회전, Shift/Swap 재배열로 정렬을 맞춥니다.
   - Analysis: 필터 체인에 효과를 추가하고 파라미터를 조절한 뒤, 실시간 프리뷰로 결과를 검증합니다.
5. **결과 활용**
   - 스크린샷/내보내기 기능으로 결과를 저장하거나, 필터 프리셋을 저장해 다른 이미지에 재사용합니다.

## 활용 예시
- 렌더링/촬영 결과물 품질 비교, QA 단계의 세부 차이 검증.
- 리터칭 전후 비교, 필드 데이터의 정렬·보정, 과학 이미지 분석.
- 대규모 필터 실험, 사용자 정의 처리 파이프라인 설계 및 검증.
