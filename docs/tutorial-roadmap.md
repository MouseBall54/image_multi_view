# 튜토리얼 로드맵

앱을 처음 접하는 사용자부터 고급 사용자까지 빠르게 적응하도록 돕는 튜토리얼 아이디어를 정리했습니다. 각 항목은 목표, 다뤄야 할 핵심 흐름, 촬영 시 주의사항을 포함합니다.

| 제목 | 다룰 내용 | 비고 |
| --- | --- | --- |

## 제작 팁

- 튜토리얼 클립은 10~20초 내외로 핵심 메시지에 집중하고 해상도를 통일합니다.
- 동일 샘플 이미지 세트를 재사용해 시각적 일관성을 유지합니다.
- 새 튜토리얼을 추가할 때마다 `src/config/tutorials.ts`에 메타데이터를 등록합니다.
- 이 문서를 체크리스트로 활용하거나 이슈 트래커에 복제해 진행 상황을 추적하세요.
