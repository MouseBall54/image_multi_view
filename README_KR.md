# compareX

데스크톱(Electron)과 웹에서 동일하게 동작하는 고급 다중 뷰 이미지 비교·분석 워크스페이스입니다. compareX는 모든 연산을 클라이언트에서 처리해 민감한 이미지를 안전하게 유지하면서, 정렬·필터링·결과 캡처까지 전문 도구 수준의 제어권을 제공합니다.

## 개요
- **세 가지 모드의 단일 워크플로** – Compare / Pinpoint / Analysis 모드를 이미지 재로딩 없이 즉시 전환할 수 있으며, 뷰포트 상태와 드래그 앤 드롭 입력, 공통 툴바가 그대로 유지됩니다.
- **로컬 우선 처리** – OpenCV.js와 UTIF 기반 TIFF 디코더, 필터 체인이 모두 브라우저 또는 Electron 런타임 안에서 실행되어 데이터가 외부로 전송되지 않습니다.
- **프로덕션급 인터페이스** – 미니맵, 사각형 줌, 회전 레벨링, 캡처 모달, 성능 지표 등 현업에서 필요한 요소를 기본 UI에 통합했습니다.

## 주요 기능
### Compare 모드
- 최대 6×4(24뷰) 레이아웃과 자동 파일명 매칭으로 대량 이미지 비교를 빠르게 구성합니다.
- 폴더 교체, 소스 스와핑, 토글 기능을 통해 한 화면에서 다양한 후보를 순환하면서 필터·회전 상태를 유지합니다.
- 숫자 입력이 가능한 공용 팬/줌, 동기화된 미니맵, 컬러 그리드 오버레이로 정량 분석을 지원합니다.

### Pinpoint 모드
- 참조 핀, 뷰어별 배율/회전, 수평·수직 레벨링 도구를 활용한 정밀 정렬 환경을 제공합니다.
- Shift / Swap 두 가지 재배치 모드로 이미지를 손쉽게 재정렬하면서 변환과 주석을 그대로 유지합니다.
- 크로스헤어 오버레이와 배율 표시로 계측 과정을 문서화하기 편리합니다.

### Analysis 모드
- 하나의 원본 이미지에 여러 필터 출력을 나란히 배치해 실험 결과를 즉시 비교합니다.
- 필터 카트에서 불러온 체인 또는 슬롯별 필터 스택을 실시간 파라미터 조절과 함께 적용할 수 있습니다.
- 라벨, 그리드, 미니맵 포함 여부를 선택해 클립보드 또는 파일(Electron)로 바로 캡처합니다.

### 필터 시스템
- 톤/히스토그램/임계값/노이즈 제거/에지/모폴로지/텍스처/주파수/컬러맵을 아우르는 60+ OpenCV 가속 필터를 지원합니다.
- 필터 카트 패널에서 드래그 정렬, 프리셋 저장·불러오기, JSON 내보내기/가져오기, 적용 전 미리보기를 제공합니다.
- 필터 성능 게이지로 예상 처리 시간과 메모리 사용량을 확인하며 파이프라인을 튜닝할 수 있습니다.

### 이미지 입력 & 정리
- 폴더 선택과 드래그 앤 드롭으로 JPEG, PNG, TIFF(UTIF 디코딩), WebP, GIF, BMP, SVG 파일을 불러올 수 있으며, 브라우저 제한으로 폴더 접근이 막힌 경우 사용자 친화적 토스트를 띄웁니다.
- 파일명(확장자 제거 옵션 포함) 기준 매칭으로 누락 파일을 신속히 파악하고 정리할 수 있습니다.
- ImageBitmap 캐시 파이프라인이 대용량 TIFF도 부드럽게 다루면서 향후 성능 추정에 필요한 해상도 메타데이터를 기록합니다.

### 내비게이션·오버레이·캡처
- 뷰포트 툴바에서 사각형 줌, 배율/좌표 입력, 미니맵·그리드·필터 라벨 전역 토글을 제공합니다.
- 캡처 모달에서 레이어별(라벨, 크로스헤어, 미니맵, 그리드) 포함 여부를 선택해 PNG로 복사/저장할 수 있습니다.
- 미니맵 위젯은 위치/크기 커스터마이징과 클릭 이동을 지원해 넓은 이미지 탐색을 돕습니다.

### 데스크톱 특화 기능
- Electron 빌드는 채널별 업데이트 피드, 진행률 토스트, 수동 재시도 버튼이 포함된 자동 업데이트 관리자를 제공합니다.
- 개발/배포 채널을 선택해 QA 단계에서 안전하게 패키지를 검증할 수 있습니다.

## 기본 사용 흐름
1. **compareX 실행** – Electron 앱 또는 Vite 개발 서버를 실행하고 상단 모드 스위처에서 Compare/Pinpoint/Analysis 중 필요한 모드를 선택합니다.
2. **이미지 가져오기** – A–Z 슬롯에 폴더를 할당하거나 폴더/파일을 워크스페이스로 드래그합니다. 파일 매칭 결과는 누락 항목을 함께 표시합니다.
3. **Compare 모드에서 전체 비교** – 동기화된 팬/줌과 레이아웃 선택기, 토글 모달, 미니맵/그리드 오버레이를 활용해 대량 이미지를 검토합니다.
4. **Pinpoint 모드로 정렬 다듬기** – 참조 핀을 배치하고 뷰어별 배율·회전을 조정하거나 레벨링 캡처로 수평선/수직선을 맞춥니다.
5. **필터 실험** – 필터 카트에서 연속 작업을 큐잉하고 파라미터를 조정한 뒤, 선택한 뷰어에 적용하거나 Analysis 모드 슬롯으로 전달합니다.
6. **결과 캡처 및 공유** – 캡처 모달에서 필요한 오버레이를 선택하고 클립보드 복사 또는 파일 저장(Electron)을 수행합니다.

## 단축키 요약
| 분류 | 조합키 | 기능 |
| --- | --- | --- |
| 모드 전환 | `Ctrl + 1` / `Ctrl + 2` / `Ctrl + 3` | Pinpoint / Analysis / Compare |
| 패널 & 오버레이 | `Ctrl + F` 컨트롤 패널, `Alt + L` 파일 리스트, `Alt + M` 미니맵, `Alt + G` 그리드, `Alt + I` 정보 패널, `Alt + R` 뷰 리셋 | 주요 UI 토글 |
| 내비게이션 | `+`, `=`, `-` 줌, `Shift` + 방향키 팬 | 빠른 뷰포트 조작 |
| 필터 라벨 | `Ctrl + L` | 뷰어 필터 배지 토글 |
| 클립보드 | `Ctrl + C` | 선택 텍스트 복사 (OS 기본 동작 유지) |

전체 단축키는 [`SHORTCUTS_GUIDE.md`](./SHORTCUTS_GUIDE.md)를 참고하세요.

## 설치 및 개발
### 사전 준비
- Node.js 18 이상, npm 9 이상 (Vite + Electron Builder 툴체인)
- WebGL 가속을 지원하는 macOS / Windows / Linux 환경

### 설치 및 실행
```bash
# 의존성 설치
npm install

# Vite 개발 서버 실행 (dev 채널)
npm run dev

# 프로덕션 플래그로 실행 (Electron 환경과 유사)
npm run dev:prod
```

### Electron 작업 흐름
```bash
# 개발 서버와 함께 Electron 실행 (핫 리로드)
npm run electron:dev

# 프로덕션 패키징
npm run electron:pack

# Windows 설치 파일 생성
npm run electron:pack:win
```
자동 업데이트는 `package.json`의 `deployTargets` 설정을 따르며, 앱 내 업데이트 패널에서 선택한 채널을 존중합니다.

### 품질 점검
```bash
# 타입 검사
npm run lint

# 프로덕션 번들링
npm run build

# 번들 미리보기
npm run preview
```

## 프로젝트 구조
| 경로 | 설명 |
| --- | --- |
| `src/App.tsx` | 모드 전환, 글로벌 툴바, 캡처 모달, 업데이트 관리자를 포함한 루트 쉘 |
| `src/modes/` | Compare / Pinpoint / Analysis 각 모드 전용 캔버스와 로직 |
| `src/components/` | 이미지 캔버스, 필터 카트, 토스트, 레이아웃 선택기, 회전 컨트롤 등 공용 UI 컴포넌트 |
| `src/utils/` | OpenCV 어댑터, UTIF 디코딩, 드래그 앤 드롭, 필터 체인 관리 등 유틸리티 |
| `src/store.ts` | 뷰어 레이아웃, 필터, 선택 상태, 토스트를 조율하는 Zustand 전역 상태 |
| `docs/` | 튜토리얼 로드맵, 필터 파라미터 제한, UX 개선 노트 등 추가 문서 |
