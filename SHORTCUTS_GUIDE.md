# compareX 단축키 가이드

## 🚀 **새로운 조합키 시스템**

텍스트 복사 기능과의 충돌을 방지하기 위해 모든 단일 키 단축키를 조합키로 변경했습니다.

### 📋 **UI 컨트롤 단축키**

| 단축키 | 기능 | 이전 |
|--------|------|------|
| **`Ctrl + F`** | Controls 패널 토글 | `F` |
| **`Alt + L`** | 파일 리스트 토글 | `L` |  
| **`Alt + M`** | 미니맵 토글 | `M` |
| **`Alt + G`** | 그리드 토글 | `G` |
| **`Alt + R`** | 뷰 리셋 | `R` |
| **`Alt + I`** | 정보 패널 토글 | `I` |

### 🎯 **모드 전환**

| 단축키 | 모드 | 이전 |
|--------|------|------|
| **`Ctrl + 1`** | Pinpoint 모드 | `1` |
| **`Ctrl + 2`** | Analysis 모드 | `2` |
| **`Ctrl + 3`** | Compare 모드 | `3` |

### 🔍 **확대/축소 제어**

| 단축키 | 기능 | 변경 사항 |
|--------|------|----------|
| **`+`** | 확대 | 단일 키로 복원 (빠른 접근) |
| **`=`** | 확대 | 단일 키로 복원 (빠른 접근) |
| **`-`** | 축소 | 단일 키로 복원 (빠른 접근) |

### 📁 **필터 관련**

| 단축키 | 기능 |
|--------|------|
| **`Ctrl + L`** | 필터 레이블 토글 |

### 🏃‍♂️ **네비게이션**

| 단축키 | 기능 |
|--------|------|
| **`Shift + ↑`** | 위로 패닝 |
| **`Shift + ↓`** | 아래로 패닝 |
| **`Shift + ←`** | 왼쪽으로 패닝 |
| **`Shift + →`** | 오른쪽으로 패닝 |

### 📄 **텍스트 복사**

| 단축키 | 기능 |
|--------|------|
| **`Ctrl + C`** | 선택된 텍스트 복사 |

## 🎉 **장점**

### ✅ **텍스트 입력 충돌 방지**
- 이제 필터 파라미터, 파일명 등을 자유롭게 타이핑 가능
- 텍스트 선택 후 `Ctrl + C`로 복사 가능

### ✅ **표준 단축키 준수**
- Windows/Linux 표준 단축키 패턴 따름
- 다른 프로그램과 일관된 사용자 경험

### ✅ **실수 방지**
- 우발적인 모드 전환이나 UI 변경 방지
- 의도적인 조합키로만 기능 실행

## 💡 **빠른 적응 팁**

1. **가장 자주 사용하는 기능부터 익히세요**
   - `Ctrl + 1/2/3`: 모드 전환
   - `Ctrl + F`: 컨트롤 패널

2. **논리적 그룹으로 기억하세요**
   - **Ctrl + 키**: 핵심 기능 (모드, 컨트롤)
   - **Alt + 키**: UI 요소 토글 (리스트, 미니맵, 그리드, 뷰 리셋)
   - **단일 키**: 자주 사용하는 기능 (확대/축소 +/-/=)

3. **텍스트 작업이 우선됩니다**
   - 단축키보다 텍스트 입력/복사가 우선
   - 충돌 없이 자유롭게 텍스트 작업 가능
